[!] 부분은 형호가 작성했기 때문에 틀릴 수도 있음. 참고용으로 보기

3. 각자 구경하다가 클래스나 div를 건드리지 않고 안의 내용만 수정하기. 다른거 건드릴 경우 복구가 힘들 수 있음.

4. 작업하고 컴퓨터를 끄기 전 꼭 git add, commit, push 까지 마치고 끄기. 그래야 서로 어디까지 작업한지 볼 수 있음. 안할 경우 한명만 독박 쓰는 거처럼 보임.

5. git add . <- 절대 사용 금지. 이상한 파일 올라가니까 본인이 건드린 파일만 올려주세요.
